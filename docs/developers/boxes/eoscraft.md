## Dependencies
### Boxes
* [`eoscraft-frontend`](eoscraft-frontend.md)

## Install
```bash
zeus unbox eoscraft
```
## Examples
### Deploy Frontend
```bash
zeus deploy frontend main
```
### Run Frontend Locally
```bash
zeus run frontend main
```
### Run Locally
```bash
zeus migrate
```
